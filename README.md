# ZoomRide - Self Drive Car Rental Platform

ZoomRide is a responsive web application for self-drive car rentals, developed using **HTML, CSS, Bootstrap 5, and Vanilla JavaScript**. It uses `localStorage` for data persistence, making it a completely client-side application.

## Features

### Core Features (Sprint 01)
- **User Authentication**: Sign Up and Login functionality.
- **Car Search**: Filter cars by Location, Type, and Fuel.
- **Car Listing**: View detailed information about available cars.
- **Booking System**: Select dates and book cars.

### Advanced Features (Sprint 02)
- **Payment Integration**: Simulated payment gateway with success/failure scenarios.
- **Admin Dashboard**: 
    - Manage Cars (Add, Edit, Delete).
    - Manage Bookings (View, Delete).
- **User Dashboard**: View personal booking history.
- **Notifications**: Toast notifications for all major actions.

## How to Run

1.  Clone or Download the repository.
2.  Open `index.html` in your web browser.
3.  **No backend setup required!**
